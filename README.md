# 📐 SD Aspect Ratio Helper

**Aspect Ratio Helper** est une extension pour [AUTOMATIC1111 Stable Diffusion WebUI](https://github.com/AUTOMATIC1111/stable-diffusion-webui) qui vous permet de définir facilement les dimensions de génération à partir d’un ratio standard.

---

## ✨ Fonctionnalités

- 🔢 Choix rapide d’un ratio dans une liste de formats standards :
  - `1:1`, `4:3`, `3:2`, `16:9`, `2:3`, `3:4`, `5:4`, `7:5`, `9:16`, `4:5`, `6:7`
  - Formats papier : `8.5x11`, `11x14`, `A4`, `A3`
- ➗ Calcul automatique des dimensions (basé sur la largeur ou la hauteur modifiée)
- ✅ Arrondi automatique à un multiple de 64 (compatible SD)
- 🖼️ Aperçu graphique du format sélectionné :
  - Cadre dessiné à l’échelle
  - Dimensions au centre
  - Icône d’orientation (H = Horizontal, V = Vertical)
  - Fond coloré selon l’orientation (bleu/rose)
- ⚠️ Avertissement si l’image est trop petite (< 256px)
- 🌍 Traduction automatique (selon WebUI) :
  - 🇫🇷 Français
  - 🇬🇧 English
  - 🇩🇪 Deutsch
  - 🇪🇸 Español
  - 🇵🇹 Português
  - 🇸🇦 العربية
